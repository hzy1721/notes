# 开发调试

## RN CLI

通过 `npx react-native` 执行命令。

- 环境
  - `doctor`：检查开发环境
  - `info`：获取环境信息
- 开发
  - `init MyApp`：初始化 MyApp 项目
- 调试
  - `start`：Metro 开发服务器
  - `devtools`：启动 DevTools
- Android
  - `run-android`：运行 Android App
  - `log-android`：查看 Android 原生日志
- iOS
  - `run-ios`：运行 iOS App
  - `log-ios`：查看 iOS 原生日志
- 生产
  - `bundle`：Metro 生产打包
