# 概述

状态管理库用于在组件之间共享状态，比如非父子（prop）、非祖先后代（context）的组件。

| 状态管理库 | 时代     | UI 库 |
| ---------- | -------- | ----- |
| Redux      | 类组件   | All   |
| Mobx       | 类组件   | React |
| Zustand    | 函数组件 | React |
| Recoil     | 函数组件 | React |
| Jotai      | 函数组件 | React |
| Valtio     | 函数组件 | React |
| Hox        | 函数组件 | React |
