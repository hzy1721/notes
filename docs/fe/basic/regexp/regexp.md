# 正则表达式

正则表达式 (Regular Expression) 由 2 部分组成：

- 模式 (pattern)
- 修饰符 (flag)

| 修饰符 | 含义               |
| ------ | ------------------ |
| `i`    | 不区分大小写       |
| `g`    | 搜索所有匹配项     |
| `m`    | 多行模式           |
| `s`    | 允许 `.` 匹配 `\n` |
| `u`    | 完整 Unicode 支持  |
| `y`    | 在指定位置搜索     |

创建正则表达式：

- `new RegExp('pattern', 'flags')`
  - 支持动态
- `/pattern/flags`
  - 静态

## 字符类

| 字符类       | 含义                                          |
| ------------ | --------------------------------------------- |
| `\d` (digit) | 数字                                          |
| `\s` (space) | 空白符：空格、`\t`、`\n`、`\v`、`\f`、`\r` 等 |
| `\w` (word)  | “单字”：字母、数字、下划线 `_`                |
| `\D`         | `\d` 反向类，非数字                           |
| `\S`         | `\S` 反向类，非空白符                         |
| `\W`         | `\w` 反向类，非字母、数字、下划线             |
| `.`          | 除 `\n` 外任何字符                            |
