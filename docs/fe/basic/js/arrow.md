# 箭头函数

ES6 引入的定义函数的新方式。

```js
const func = (arg1, arg2) => {
  // ...
};
```

- 没有 `this`
  - 调用时的 `this` 为声明时外部的 `this`
- 没有 `arguments`
  - 可以用 rest 参数代替
- 不能作为构造函数
  - 使用 `new` 会报错
- 不能作为生成器函数
  - 函数内不能使用 `yield`

优点：

- 用箭头 `=>` 代替 `function` 关键字，语法更简洁
- 只有一个参数可以省略圆括号，函数体只有一条语句可以省略花括号，简化了回调函数的写法
