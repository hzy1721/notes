# 常量与变量

## 常量

用 `let` 声明的 “变量” 默认是不可变的（也就是常量）。

```rs
let x = 5;
```

允许重复声明同名变量，后面的会覆盖前面的，从而实现对变量值和类型的修改，并在修改后保持其不可变的特性。

```rs
let x = 5;
let x = x + 1;
```

用 `const` 声明常量，显式标注类型，初始值是常量表达式（编译时计算），支持全局作用域。

```rs
const ONE_HOUR_SECONDS: u32 = 60 * 60;
```

## 变量

在 `let` 后面加上 `mut` 来声明变量。

```rs
let mut x = 5;
```
