# 概述

## 安装

安装 `rustup`、`rustc`、`cargo` 等工具：

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

更新：

```sh
rustup update
```

## VSCode 插件

`rust-analyzer`

## cargo

| 用途             | 命令          |
| ---------------- | ------------- |
| 创建项目         | cargo new     |
| 编译             | cargo build   |
| 运行             | cargo run     |
| 测试             | cargo test    |
| 生成文档         | cargo doc     |
| 发布到 crates.io | cargo publish |

## 官方 registry

https://crates.io/
