## 格式化输出

| 转义序列 | 输出类型 |
| -------- | -------- |
| %s       | 字符串   |
| %i/%d    | 整数     |
| %f       | 浮点数   |
| %o/%0    | 对象     |
| %c       | CSS 样式 |
