# TCP 和 UDP 的区别

|                  | TCP                  | UDP                                |
| ---------------- | -------------------- | ---------------------------------- |
| 是否面向连接     | 面向连接             | 无连接                             |
| 是否可靠         | 提供可靠交付         | 尽最大努力交付                     |
| 报文大小         | 面向字节流，大小不等 | 交付完整报文                       |
| 拥塞控制         | 有，慢开始/拥塞避免  | 无，能够保证恒定的发送速率         |
| 是否支持多方通信 | 只能是一对一         | 支持一对一、一对多、多对一、多对多 |
| 性能             | 开销大               | 开销小                             |
| 应用层协议       | HTTP/TLS/FTP/SMTP    | DNS/DHCP                           |
