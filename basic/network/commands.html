常用命令 #

ping #

发送 ICMP ECHO_REQUEST 报文。用于验证网络连通性和连接质量。

sh
ping baidu.com

ifconfig #

显示和配置网络适配器的信息。

sh
ifconfig

traceroute #

查看数据包经过的 IP、响应时间。

sh
traceroute baidu.com

netstat #

显示网络状态。

sh
netstat
